# V4.2 — 3D Full Scale Solar System Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Přepnout layout z "schematický horizontal seřazený pohled" (planety na ose X od `-1500..+200`) na **3D solar system s planetami v kruhových orbitách kolem Slunce v origin (0,0,0)**. Slunce uprostřed, planety obíhají v XZ rovině s real proporcí vzdáleností (compressed sqrt mapping).

**Architecture:**
- Slunce přesunuto do origin. PLANETS data: vyhodit `xPosition`, přidat `orbitRadius`, `orbitalPeriodSec`, `initialPhaseRad`.
- Nový modul `planetOrbits.js` — pure-math `orbitalPosition(planet, elapsed) → {x, y, z}` + `updatePlanetOrbits(anchors, elapsed)`.
- `planetAnchors.js` — pozice anchora dynamic (set každý frame přes `updatePlanetOrbits`).
- Kamera: výchozí (0, 2500, 4500) hledí na origin, vidí celou soustavu z mírného úhlu.
- `solarWind.js` — Sun je teď v origin (0,0,0), targets recompute z aktuální anchor.position (cached jen `localOffset`, pos = anchor.pos + offset).
- Měsíce zachovávají Kepler orbits kolem rodiče (anchor parent-child relation drží).

**Tech Stack:** Three.js Object3D, vanilla JS, žádný shader change.

## AU → Display Radius mapping

Konstanta v `planetOrbits.js`:
```js
// Real AU vzdálenosti planet:
//   Mercury 0.39 · Venus 0.72 · Earth 1.00 · Mars 1.52
//   Jupiter 5.20 · Saturn 9.55 · Uranus 19.20 · Neptune 30.05
//
// Mapping: displayR = 1100 + 200 × sqrt(au)
//   Mercury 1100+125 = 1225  (Sun radius 995, buffer 230)
//   Venus   1100+170 = 1270
//   Earth   1100+200 = 1300
//   Mars    1100+247 = 1347
//   Jupiter 1100+456 = 1556
//   Saturn  1100+618 = 1718
//   Uranus  1100+877 = 1977
//   Neptune 1100+1097 = 2197
//
// To dává compressed inner-planet cluster (Mer/Ven/Ear/Mars rozsah 122 units)
// + outer planety stále viditelně dál (Jup/Sat/Ura/Nep rozsah 641 units).
export function auToDisplayRadius(au) {
  return 1100 + 200 * Math.sqrt(au);
}
```

## Orbital periods (compressed)

Real periods:
- Mercury 88 days · Venus 225 · Earth 365 · Mars 687
- Jupiter 4333 · Saturn 10759 · Uranus 30687 · Neptune 60190

Compressed: divisor 365/10 = 36.5 (Earth = 10 sec, jako V1 rotation).
- Mercury 88/36.5 = 2.4s
- Venus 225/36.5 = 6.2s
- Earth 365/36.5 = 10s
- Mars 687/36.5 = 18.8s
- Jupiter 4333/36.5 = 118.7s (~2 min)
- Saturn 10759/36.5 = 294.8s (~5 min)
- Uranus 30687/36.5 = 840.7s (~14 min)
- Neptune 60190/36.5 = 1648.4s (~27 min)

Outer planety se prakticky nehnou v běžné session — to je realistické. Pro vizuální pohyb v rozumném čase: další divisor `/3` pro outer (>=Jupiter):
- Jupiter ~40s · Saturn ~98s · Uranus ~280s · Neptune ~550s

V plánu jdeme s `/36.5` čistě (real ratios) + `/3` skip pro outer = jednoduchá rule:
```js
const periodSec = realDays / 36.5 / (au >= 5 ? 3 : 1);
```

Anebo definovat `orbitalPeriodSec` přímo per-planet v PLANETS pro jasnost.

```js
// planetOrbits — pure-math helpers pro 3D orbity planet kolem Slunce.
// Slunce je v origin (0,0,0), planety obíhají v XZ rovině (Y=0).
// Žádný eccentricity (kruhové orbity pro V4.2 — eccentricity přijde s V4.3 sim).

// Real AU vzdálenosti:
//   Mercury 0.39 · Venus 0.72 · Earth 1.00 · Mars 1.52
//   Jupiter 5.20 · Saturn 9.55 · Uranus 19.20 · Neptune 30.05
// Mapping displayR = 1100 + 200×sqrt(au) — Sun radius 995, buffer 230 u Mercuru,
// outer planety stále viditelně dál bez extrémní compresse.
export function auToDisplayRadius(au) {
  return 1100 + 200 * Math.sqrt(au);
}

/**
 * Pozice planety v čase elapsed (sekund). Kruhová orbita v XZ rovině.
 * @param {{ orbitRadius: number, orbitalPeriodSec: number, initialPhaseRad: number }} planet
 * @param {number} elapsed
 * @returns {{ x: number, y: number, z: number }}
 */
export function orbitalPosition(planet, elapsed) {
  if (planet.orbitRadius === 0) return { x: 0, y: 0, z: 0 };
  const omega = (2 * Math.PI) / planet.orbitalPeriodSec;
  const theta = planet.initialPhaseRad + omega * elapsed;
  return {
    x: planet.orbitRadius * Math.cos(theta),
    y: 0,
    z: planet.orbitRadius * Math.sin(theta),
  };
}
```

Přidej na konec souboru (mimo pure-math, používá Object3D):

```js
/**
 * Aktualizuje pozici všech planet anchors podle aktuálního času.
 * Sun (orbitRadius=0) zůstává v origin.
 *
 * @param {Object} anchors — { [planetId]: THREE.Object3D }
 * @param {Array} planets — PLANETS array
 * @param {number} elapsed
 */
export function updatePlanetOrbits(anchors, planets, elapsed) {
  for (const p of planets) {
    const a = anchors[p.id];
    if (!a) continue;
    const pos = orbitalPosition(p, elapsed);
    a.position.set(pos.x, pos.y, pos.z);
  }
}
```

```js
// orbitLines — tenké šedé kruhy v XZ rovině per planeta + dwarf,
// vyznačují trajektorii. Kruhy jsou statické mesh-y (LineLoop), nezávislé
// na simMode (radius reflektuje aktuální orbitRadius × spaceFactor).

import * as THREE from 'three';
import { PLANETS } from './planets.js';

const SEGMENTS = 128;

/**
 * Vytvoří LineLoop per planetu (kromě Sun) v scene.
 * @param {THREE.Scene} scene
 * @returns {Array<{ planetId: string, line: THREE.LineLoop, baseRadius: number }>}
 */
export function createOrbitLines(scene) {
  const lines = [];
  const material = new THREE.LineBasicMaterial({
    color: 0x666666,
    transparent: true,
    opacity: 0.35,
    depthWrite: false,
  });
  for (const p of PLANETS) {
    if (p.orbitRadius === 0) continue; // Sun
    const positions = new Float32Array((SEGMENTS + 1) * 3);
    for (let i = 0; i <= SEGMENTS; i++) {
      const theta = (i / SEGMENTS) * Math.PI * 2;
      positions[i * 3]     = p.orbitRadius * Math.cos(theta);
      positions[i * 3 + 1] = 0;
      positions[i * 3 + 2] = p.orbitRadius * Math.sin(theta);
    }
    const geom = new THREE.BufferGeometry();
    geom.setAttribute('position', new THREE.BufferAttribute(positions, 3));
    const line = new THREE.Line(geom, material);
    line.userData.bodyId = p.id;
    scene.add(line);
    lines.push({ planetId: p.id, line, baseRadius: p.orbitRadius });
  }
  return lines;
}

/**
 * Update radius pokud se změnil simMode space factor — překreslí všechny kruhy.
 */
export function updateOrbitRadii(lines, spaceFactor) {
  for (const entry of lines) {
    const r = entry.baseRadius * spaceFactor;
    const positions = entry.line.geometry.attributes.position.array;
    for (let i = 0; i <= SEGMENTS; i++) {
      const theta = (i / SEGMENTS) * Math.PI * 2;
      positions[i * 3]     = r * Math.cos(theta);
      positions[i * 3 + 2] = r * Math.sin(theta);
    }
    entry.line.geometry.attributes.position.needsUpdate = true;
  }
}
```

```js
// simMode — 4-mode space/time scaling per V4 spec.
// VÝCHOZÍ:    space×1, time×1 (compressed both, current default)
// PROPORČNÍ:  space×REAL, time×1 (real distance ratios, kamera musí umět zoom out)
// REÁLNÝ ČAS: space×1, time×REAL (real Kepler periody)
// SIMULACE:   space×REAL, time×REAL (fyzikálně přesné)
//
// Real factors = "rozsahový" multiplikátor proti VÝCHOZÍ:
//   space: real distance Mercury 0.39 AU vs Neptune 30 AU = ratio 77.
//          VÝCHOZÍ má sqrt mapping (Mercury/Neptune ratio ~1.8). Real = 77/1.8 = 42.
//          Aplikovat 42× škála na vnější planety + zachovat Sun radius.
//          Implementace: real = pure linear AU mapping. computeRealRadius(planet).
//   time:  real period ratio. Mercury 88d, Neptune 60190d → 684× pomalejší.
//          VÝCHOZÍ má outer /3 divisor. Real = vyhodit ten /3 + zvětšit timescale 36.5.

const MODES = {
  default:     { id: 'default',     label: 'VÝCHOZÍ',    space: 'compressed', time: 'compressed' },
  proportional:{ id: 'proportional',label: 'PROPORČNÍ',  space: 'real',       time: 'compressed' },
  realtime:    { id: 'realtime',    label: 'REÁLNÝ ČAS', space: 'compressed', time: 'real' },
  simulation:  { id: 'simulation',  label: 'SIMULACE',   space: 'real',       time: 'real' },
};

let currentMode = MODES.default;
let timeSpeed = 1.0; // user slider 0.1..1000

const listeners = [];

export function getMode() { return currentMode; }
export function setMode(id) {
  const m = MODES[id];
  if (!m) return;
  currentMode = m;
  for (const cb of listeners) cb(currentMode);
}
export function setTimeSpeed(s) { timeSpeed = Math.max(0.1, Math.min(1000, s)); }
export function getTimeSpeed() { return timeSpeed; }

export function isSpaceReal() { return currentMode.space === 'real'; }
export function isTimeReal() { return currentMode.time === 'real'; }

/**
 * @param {{ orbitRadius: number, orbitRadiusReal?: number }} planet
 * @returns {number} aktuální display radius
 */
export function getOrbitRadiusFor(planet) {
  if (currentMode.space === 'real' && planet.orbitRadiusReal !== undefined) {
    return planet.orbitRadiusReal;
  }
  return planet.orbitRadius;
}

/**
 * @param {{ orbitalPeriodSec: number, orbitalPeriodSecReal?: number }} planet
 * @returns {number} aktuální period (po time speed slider)
 */
export function getPeriodFor(planet) {
  const base = (currentMode.time === 'real' && planet.orbitalPeriodSecReal !== undefined)
    ? planet.orbitalPeriodSecReal
    : planet.orbitalPeriodSec;
  return base / timeSpeed;
}

export function onChange(cb) { listeners.push(cb); }

export const ALL_MODES = Object.values(MODES);
```

```js
// formationIntro — controller pro 6-beat narrativ vzniku sluneční soustavy.
// Vychází z V4 spec (Nice model + protoplanetary disk research).

import { phaseAt } from './animation.js';
import { PLANETS } from './planets.js';
import { PHASE } from './phase.js';

const CLOUD_PARTICLE_COUNT = 18000;
const CLOUD_MAX_RADIUS = 3500; // ~vzdálenost Neptune (2200) + buffer
const CLOUD_MIN_RADIUS = 200;

let _cloudInitialized = false;

function initCloud(pool) {
  if (_cloudInitialized) return;
  _cloudInitialized = true;
  // Spawn 18k teček v sférické distribuci (random pozice v shell mezi MIN..MAX).
  // Šedo-modrá barva (protoplanetární prach), low alpha pro éterický feel.
  const idle = pool.takeIdleIndices(CLOUD_PARTICLE_COUNT);
  for (let k = 0; k < idle.length; k++) {
    const i = idle[k];
    const theta = Math.random() * Math.PI * 2;
    const phi = Math.acos(2 * Math.random() - 1);
    const r = CLOUD_MIN_RADIUS + Math.random() * (CLOUD_MAX_RADIUS - CLOUD_MIN_RADIUS);
    pool.position[3*i]     = r * Math.sin(phi) * Math.cos(theta);
    pool.position[3*i + 1] = r * Math.cos(phi) * 0.3; // slightly flatter (disk-shape)
    pool.position[3*i + 2] = r * Math.sin(phi) * Math.sin(theta);
    pool.color[3*i]     = 0.45 + Math.random() * 0.15;
    pool.color[3*i + 1] = 0.5  + Math.random() * 0.15;
    pool.color[3*i + 2] = 0.6  + Math.random() * 0.2;
    pool.alpha[i] = 0.7;
    pool.size[i] = 4.5;
    pool.phase[i] = 99; // custom: cloud particle (mimo standard PHASE enum)
    pool.owner[i] = -1;
    pool.localOffset[3*i]     = pool.position[3*i];
    pool.localOffset[3*i + 1] = pool.position[3*i + 1];
    pool.localOffset[3*i + 2] = pool.position[3*i + 2];
  }
  pool.cloudIndices = idle;
  pool.flushAll();
}

/**
 * Beat 1 (0-3s): cloud rotuje kolem origin (slow orbit ~30s/full).
 * Beat 2 (3-6s): cloud kolapsu — všechny particles kondenzují k středu (lerp position k 0).
 */
export function updateFormationIntro(pool, currentTime, dt) {
  const ph = phaseAt(currentTime);
  if (!ph) return;

  if (ph.id === 'beat1_cloud') {
    initCloud(pool);
    // Slow rotation kolem Y osy (~30s/full)
    const omega = (2 * Math.PI) / 30;
    const angle = omega * dt;
    const c = Math.cos(angle), s = Math.sin(angle);
    for (const i of pool.cloudIndices || []) {
      const x = pool.position[3*i], z = pool.position[3*i + 2];
      pool.position[3*i]     = x * c - z * s;
      pool.position[3*i + 2] = x * s + z * c;
    }
    pool.posAttr.needsUpdate = true;
    return;
  }

  if (ph.id === 'beat2_collapse') {
    initCloud(pool); // pojistka pokud Beat 1 přeskočil
    const t = (currentTime - ph.start) / (ph.end - ph.start); // 0..1
    const collapseFactor = 1 - t * t; // smooth ease (1 → 0)
    for (const i of pool.cloudIndices || []) {
      const lx = pool.localOffset[3*i];
      const ly = pool.localOffset[3*i + 1];
      const lz = pool.localOffset[3*i + 2];
      pool.position[3*i]     = lx * collapseFactor;
      pool.position[3*i + 1] = ly * collapseFactor;
      pool.position[3*i + 2] = lz * collapseFactor;
      // Fade-out u centra ať se nesplynou v jeden bod
      pool.alpha[i] = collapseFactor * 0.7 + 0.05;
    }
    pool.posAttr.needsUpdate = true;
    pool.alphaAttr.needsUpdate = true;
    return;
  }

  // Beat 3+: cloud zmizí (alpha 0)
  if (pool.cloudIndices) {
    for (const i of pool.cloudIndices) pool.alpha[i] = 0;
    pool.alphaAttr.needsUpdate = true;
  }
}

export function resetFormationIntro() {
  _cloudInitialized = false;
}
```
