> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Přepnout rendering planet/měsíců/Slunce z `bodyMesh.js` (flat-face triangles) na **`voxelTile.js` (InstancedMesh hexagonálních 3D tiles s real Lambertian lightingem)** — minecraft voxel estetika s denní/noční stranou.

**Architecture:** Každé tělo je `THREE.Mesh` s `InstancedBufferGeometry` (base hexagon = 6-triangle fan) a `ShaderMaterial`. Per-instance atributy: pozice (sphere surface), normála, tangent frame, barva (sampled z textury). Vertex shader transformuje hexagon do tangent frame pro každou instanci; fragment shader aplikuje Lambertian diffuse podle směru od Slunce. Saturnův prstenec = `THREE.RingGeometry` child Saturnova anchoru (ekvatoriální rovina). Picker meshes přesunuty na Layer 1 (neviditelné v default camera).

**Tech Stack:** Three.js 0.170 (InstancedBufferGeometry, RingGeometry, Layers), vanilla GLSL ES 300, node:test.

```js
// lighting — sdílený setup pro směrové sluneční osvětlení.
// Sun je `THREE.DirectionalLight` umístěný u pozice Slunce, svítí k původu.
// Pro voxel shadery exportuje uniform `uSunPos` (world space), ze kterého
// vertex shader vypočítá directional diffuse per fragment.

import * as THREE from 'three';

/**
 * Vytvoří `THREE.Uniform`-like object { value: Vector3 } pro sun world position.
 * Bude shared napříč všemi voxel shadery (shader sám vypočítá dir per vertex).
 */
export function createSunUniform(sunPos) {
  return { value: new THREE.Vector3(sunPos.x, sunPos.y, sunPos.z) };
}

/**
 * Pure-math helper: jednotkový vektor směřující z `point` k `sunPos`.
 * Používá se v testech + potenciálně pro CPU lighting check.
 */
export function computeSunDirection(point, sunPos) {
  const dx = sunPos.x - point.x;
  const dy = sunPos.y - point.y;
  const dz = sunPos.z - point.z;
  const len = Math.sqrt(dx * dx + dy * dy + dz * dz) || 1;
  return { x: dx / len, y: dy / len, z: dz / len };
}

/**
 * Připraví scénu: DirectionalLight od Sun pozice, AmbientLight slabá.
 * Vrátí objekty pro pozdější manipulaci.
 */
export function setupSceneLighting(scene, sunPos) {
  const ambient = new THREE.AmbientLight(0xffffff, 0.08);
  scene.add(ambient);

  const sun = new THREE.DirectionalLight(0xffffff, 2.5);
  sun.position.set(sunPos.x, sunPos.y, sunPos.z);
  sun.target.position.set(0, 0, 0);
  scene.add(sun);
  scene.add(sun.target);

  return { ambient, sun };
}
```

```js
// voxelTile — staví InstancedMesh pro tělo: base hexagon geometry + per-instance
// transforms do tangent frame na sféře + per-instance color sampled z textury.
// Shader Lambertian diffuse vůči uSunPos uniform (vypočítá se dir per vertex).

import * as THREE from 'three';
import { icosphereRaw } from './geometry.js';
import { sampleColorPoleSafe, sphericalUV } from './textureUtils.js';

export const VOXEL_VERTEX_SHADER = /* glsl */ `
  attribute vec3 aInstancePos;    // kde je tile na sféře (radius × dir)
  attribute vec3 aInstanceNormal; // tile normál (= jednotkový radial ven)
  attribute vec3 aInstanceTangent;// tangent v tangent plane (pro rotaci hex)
  attribute vec3 aInstanceColor;  // per-tile RGB

  uniform vec3 uSunPos;
  uniform float uAmbient;

  varying vec3 vColor;
  varying float vLight;

  void main() {
    // Tile local space: hexagon v XY rovině (Z = normal).
    // Umístíme ho do tangent frame instance:
    //   bitangent = normal × tangent
    //   world_offset = position.x * tangent + position.y * bitangent + aInstancePos
    vec3 bitangent = cross(aInstanceNormal, aInstanceTangent);
    vec3 worldOffset = position.x * aInstanceTangent
                     + position.y * bitangent
                     + aInstancePos;

    vec4 worldPos = modelMatrix * vec4(worldOffset, 1.0);
    vec3 worldNormal = normalize((modelMatrix * vec4(aInstanceNormal, 0.0)).xyz);

    vec3 sunDir = normalize(uSunPos - worldPos.xyz);
    float diffuse = max(0.0, dot(worldNormal, sunDir));
    vLight = uAmbient + (1.0 - uAmbient) * diffuse;
    vColor = aInstanceColor;

    gl_Position = projectionMatrix * viewMatrix * worldPos;
  }
`;

export const VOXEL_FRAGMENT_SHADER = /* glsl */ `
  varying vec3 vColor;
  varying float vLight;

  void main() {
    gl_FragColor = vec4(vColor * vLight, 1.0);
  }
`;
```

```js
/**
 * Base hexagon geometry v XY rovině (Z = 0).
 * Center vrchol + 6 rohů na kružnici. 6 trojúhelníků fan.
 *
 * @param {number} radius — poloměr kružnice (circumradius)
 * @returns {{ positions: Float32Array, indices: Uint16Array }}
 */
export function buildHexagonGeometry(radius) {
  const positions = new Float32Array(7 * 3); // 7 verts × xyz
  const indices = new Uint16Array(6 * 3);    // 6 tris × 3

  // Center
  positions[0] = 0;
  positions[1] = 0;
  positions[2] = 0;

  // 6 rohů na kružnici
  for (let i = 0; i < 6; i++) {
    const angle = (i / 6) * Math.PI * 2;
    positions[(i + 1) * 3 + 0] = Math.cos(angle) * radius;
    positions[(i + 1) * 3 + 1] = Math.sin(angle) * radius;
    positions[(i + 1) * 3 + 2] = 0;
  }

  // 6 trojúhelníků: center + corner[i] + corner[i+1]
  for (let i = 0; i < 6; i++) {
    const next = (i + 1) % 6 + 1;
    indices[i * 3 + 0] = 0;
    indices[i * 3 + 1] = i + 1;
    indices[i * 3 + 2] = next;
  }

  return { positions, indices };
}
```

## Task 4 — computeTangentFrame pure math

Pro každý vrchol na sféře potřebujeme tangent frame (normal + tangent + bitangent). Normal = radial (= normalized vertex pozice). Tangent = libovolný vektor kolmý na normal — použijeme projekci globálního Y (nebo X pokud normal = ±Y).

```js
const _UP = [0, 1, 0];
const _FORWARD = [0, 0, 1];

/**
 * Tangent frame pro vrchol na sféře.
 * normal = normalized(vertex)
 * tangent = projection globálního UP na tangent plane, normalizováno.
 *   Pokud normal ≈ UP (pole), použije FORWARD jako fallback.
 *
 * @param {number[]} vertex — [x, y, z]
 * @returns {{ normal: number[], tangent: number[] }} oba jednotkové
 */
export function computeTangentFrame(vertex) {
  const len = Math.sqrt(vertex[0] ** 2 + vertex[1] ** 2 + vertex[2] ** 2) || 1;
  const normal = [vertex[0] / len, vertex[1] / len, vertex[2] / len];

  // Fallback pokud normal je blízko Y osy (pole)
  const dotUp = Math.abs(normal[1]);
  const ref = dotUp > 0.99 ? _FORWARD : _UP;

  // tangent = ref - (ref · normal) * normal, pak normalize
  const d = ref[0] * normal[0] + ref[1] * normal[1] + ref[2] * normal[2];
  let tx = ref[0] - d * normal[0];
  let ty = ref[1] - d * normal[1];
  let tz = ref[2] - d * normal[2];
  const tlen = Math.sqrt(tx * tx + ty * ty + tz * tz) || 1;
  return {
    normal,
    tangent: [tx / tlen, ty / tlen, tz / tlen],
  };
}
```

Přidej na konec `voxelTile.js` (používá už existující exports):

```js
/**
 * Postaví InstancedMesh pro tělo.
 *
 * @param {ImageData} imageData — textura pro per-tile color sampling
 * @param {number} radius — poloměr tělesa (planety/měsíce)
 * @param {number} minVertices — icosphere subdivision level volba (10242 = L5, 40962 = L6)
 * @param {Object} uniforms — shared shader uniforms, nejdůležitější `{ uSunPos: {value: Vector3} }`
 * @returns {THREE.Mesh}
 */
export function buildVoxelTiles(imageData, radius, minVertices, uniforms) {
  const { vertices } = icosphereRaw(minVertices);
  const tileCount = vertices.length;

  // Tile velikost — 4×r/sqrt(N) pro 1× coverage (dots touching, no overlap).
  // Pro icosphere N vrcholů na jednotkové kouli: mean neighbor distance ≈ 2/sqrt(N).
  // Hexagon circumradius = neighbor_distance / 2 → radius × 2/sqrt(N) / 2 = radius/sqrt(N).
  // Malý násobitel (0.58) aby se dotýkaly vertexy hex, ne strany.
  const tileRadius = radius * 0.58 / Math.sqrt(tileCount / 40000);
  const hexGeom = buildHexagonGeometry(tileRadius);

  const geom = new THREE.InstancedBufferGeometry();
  geom.setAttribute('position', new THREE.BufferAttribute(hexGeom.positions, 3));
  geom.setIndex(new THREE.BufferAttribute(hexGeom.indices, 1));

  // Per-instance atributy
  const instPos = new Float32Array(tileCount * 3);
  const instNormal = new Float32Array(tileCount * 3);
  const instTangent = new Float32Array(tileCount * 3);
  const instColor = new Float32Array(tileCount * 3);

  for (let i = 0; i < tileCount; i++) {
    const v = vertices[i];
    const { normal, tangent } = computeTangentFrame(v);

    // Instance position = vertex scaled to body radius
    instPos[i * 3 + 0] = normal[0] * radius;
    instPos[i * 3 + 1] = normal[1] * radius;
    instPos[i * 3 + 2] = normal[2] * radius;

    instNormal[i * 3 + 0] = normal[0];
    instNormal[i * 3 + 1] = normal[1];
    instNormal[i * 3 + 2] = normal[2];

    instTangent[i * 3 + 0] = tangent[0];
    instTangent[i * 3 + 1] = tangent[1];
    instTangent[i * 3 + 2] = tangent[2];

    // Sample color z textury pomocí spherical UV
    const [u, v_] = sphericalUV(normal[0], normal[1], normal[2], 1);
    const [cr, cg, cb] = sampleColorPoleSafe(imageData, u, v_);
    instColor[i * 3 + 0] = cr;
    instColor[i * 3 + 1] = cg;
    instColor[i * 3 + 2] = cb;
  }

  geom.setAttribute('aInstancePos', new THREE.InstancedBufferAttribute(instPos, 3));
  geom.setAttribute('aInstanceNormal', new THREE.InstancedBufferAttribute(instNormal, 3));
  geom.setAttribute('aInstanceTangent', new THREE.InstancedBufferAttribute(instTangent, 3));
  geom.setAttribute('aInstanceColor', new THREE.InstancedBufferAttribute(instColor, 3));
  geom.instanceCount = tileCount;

  const material = new THREE.ShaderMaterial({
    uniforms: {
      uSunPos: uniforms.uSunPos,
      uAmbient: { value: 0.12 },
    },
    vertexShader: VOXEL_VERTEX_SHADER,
    fragmentShader: VOXEL_FRAGMENT_SHADER,
  });

  return new THREE.Mesh(geom, material);
}

/**
 * Self-emissive varianta pro Slunce — nepoužívá lighting, vždy 100% barva.
 * Stejné per-instance attributes, jen jiný shader.
 */
export function buildSunVoxelTiles(imageData, radius, minVertices) {
  const { vertices } = icosphereRaw(minVertices);
  const tileCount = vertices.length;
  const tileRadius = radius * 0.58 / Math.sqrt(tileCount / 40000);
  const hexGeom = buildHexagonGeometry(tileRadius);

  const geom = new THREE.InstancedBufferGeometry();
  geom.setAttribute('position', new THREE.BufferAttribute(hexGeom.positions, 3));
  geom.setIndex(new THREE.BufferAttribute(hexGeom.indices, 1));

  const instPos = new Float32Array(tileCount * 3);
  const instNormal = new Float32Array(tileCount * 3);
  const instTangent = new Float32Array(tileCount * 3);
  const instColor = new Float32Array(tileCount * 3);

  for (let i = 0; i < tileCount; i++) {
    const v = vertices[i];
    const { normal, tangent } = computeTangentFrame(v);
    instPos[i * 3] = normal[0] * radius;
    instPos[i * 3 + 1] = normal[1] * radius;
    instPos[i * 3 + 2] = normal[2] * radius;
    instNormal[i * 3] = normal[0];
    instNormal[i * 3 + 1] = normal[1];
    instNormal[i * 3 + 2] = normal[2];
    instTangent[i * 3] = tangent[0];
    instTangent[i * 3 + 1] = tangent[1];
    instTangent[i * 3 + 2] = tangent[2];
    const [u, v_] = sphericalUV(normal[0], normal[1], normal[2], 1);
    const [cr, cg, cb] = sampleColorPoleSafe(imageData, u, v_);
    instColor[i * 3] = cr;
    instColor[i * 3 + 1] = cg;
    instColor[i * 3 + 2] = cb;
  }

  geom.setAttribute('aInstancePos', new THREE.InstancedBufferAttribute(instPos, 3));
  geom.setAttribute('aInstanceNormal', new THREE.InstancedBufferAttribute(instNormal, 3));
  geom.setAttribute('aInstanceTangent', new THREE.InstancedBufferAttribute(instTangent, 3));
  geom.setAttribute('aInstanceColor', new THREE.InstancedBufferAttribute(instColor, 3));
  geom.instanceCount = tileCount;

  const SUN_VERTEX = /* glsl */ `
    attribute vec3 aInstancePos;
    attribute vec3 aInstanceNormal;
    attribute vec3 aInstanceTangent;
    attribute vec3 aInstanceColor;
    varying vec3 vColor;
    void main() {
      vec3 bitangent = cross(aInstanceNormal, aInstanceTangent);
      vec3 worldOffset = position.x * aInstanceTangent + position.y * bitangent + aInstancePos;
      vColor = aInstanceColor;
      gl_Position = projectionMatrix * modelViewMatrix * vec4(worldOffset, 1.0);
    }
  `;
  const SUN_FRAG = /* glsl */ `
    varying vec3 vColor;
    void main() {
      gl_FragColor = vec4(vColor, 1.0);
    }
  `;

  const material = new THREE.ShaderMaterial({
    vertexShader: SUN_VERTEX,
    fragmentShader: SUN_FRAG,
  });

  return new THREE.Mesh(geom, material);
}
```

- [ ] **Step 7.2: Implementuj `src/saturnRing.js`**

```js
// saturnRing — real 3D RingGeometry v ekvatoriální rovině Saturnu.
// Per-face barva sampled z saturn_ring.png podle radiální pozice t.

import * as THREE from 'three';

/**
 * Pure sampler — vrátí [r, g, b, a] 0..1 pro radiální t (0 = inner, 1 = outer).
 * Sampling z prostředního řádku obrazu (ring textura je 1D radiální).
 */
export function sampleRingColor(imageData, t) {
  const { data, width, height } = imageData;
  const py = Math.floor(height / 2);
  const px = Math.min(width - 1, Math.max(0, Math.floor(t * width)));
  const idx = (py * width + px) * 4;
  return [data[idx] / 255, data[idx + 1] / 255, data[idx + 2] / 255, data[idx + 3] / 255];
}

/**
 * Staví Saturnův prstenec jako RingGeometry s per-vertex colors.
 * Ring je v XY rovině (Z=0), child Saturn anchoru dědí axial tilt.
 *
 * @param {ImageData} ringImageData
 * @param {number} innerRadius
 * @param {number} outerRadius
 * @param {number} segments — angular segments (default 128)
 * @param {number} radialSegments — radial subdivisions (default 8)
 */
export function buildSaturnRing(ringImageData, innerRadius, outerRadius, segments = 128, radialSegments = 8) {
  const geom = new THREE.RingGeometry(innerRadius, outerRadius, segments, radialSegments);

  // Per-vertex color: RingGeometry UV.y odpovídá radiální pozici
  const posArr = geom.attributes.position;
  const colorArr = new Float32Array(posArr.count * 3);
  const alphaArr = new Float32Array(posArr.count);

  for (let i = 0; i < posArr.count; i++) {
    const x = posArr.getX(i);
    const y = posArr.getY(i);
    const r = Math.sqrt(x * x + y * y);
    const t = (r - innerRadius) / (outerRadius - innerRadius);
    const [cr, cg, cb, ca] = sampleRingColor(ringImageData, t);
    colorArr[i * 3] = cr;
    colorArr[i * 3 + 1] = cg;
    colorArr[i * 3 + 2] = cb;
    alphaArr[i] = ca;
  }
  geom.setAttribute('color', new THREE.BufferAttribute(colorArr, 3));
  geom.setAttribute('ringAlpha', new THREE.BufferAttribute(alphaArr, 1));

  const material = new THREE.ShaderMaterial({
    uniforms: {},
    vertexShader: /* glsl */ `
      attribute float ringAlpha;
      varying vec3 vColor;
      varying float vAlpha;
      void main() {
        vColor = color;
        vAlpha = ringAlpha;
        gl_Position = projectionMatrix * modelViewMatrix * vec4(position, 1.0);
      }
    `,
    fragmentShader: /* glsl */ `
      varying vec3 vColor;
      varying float vAlpha;
      void main() {
        gl_FragColor = vec4(vColor, vAlpha);
      }
    `,
    transparent: true,
    side: THREE.DoubleSide,
    vertexColors: true,
  });

  const mesh = new THREE.Mesh(geom, material);
  // RingGeometry ve výchozím stavu leží v XY. Saturnův equator je v XZ (po applied tilt). Musíme rotovat:
  mesh.rotation.x = Math.PI / 2;
  return mesh;
}
```
